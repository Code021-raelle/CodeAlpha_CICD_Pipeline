# CodeAlpha CI/CD Pipeline Project

## Project Overview

This project demonstrates a complete CI/CD pipeline for a containerized Node.js application.

The application is automatically built, containerized using Docker, and deployed to a cloud platform whenever code changes are pushed to GitHub.

## Technologies Used

- Node.js
- Express.js
- Docker
- GitHub Actions
- Docker Hub
- Render

## CI/CD Workflow

Developer Push
→ GitHub Repository
→ GitHub Actions Pipeline
→ Docker Image Build
→ Docker Hub Registry
→ Render Deployment

## Features

- Automated CI/CD pipeline
- Docker containerization
- Continuous deployment
- Cloud hosting
- Deployment monitoring
